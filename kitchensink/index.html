Spartan UI KIT

Small UI kit ~6kb in size (without filtering and sorting) for application development.

Default font is Roboto Latin-ext implemented into css as @import.

Icons can be utf-8 characters ‚ûú copypastecharacter.com or utf-8 emoji.

üåö Dark mode

Works only when user has dark theme enabled through global OS theme settings. This features uses prefers-color-scheme to determine if dark theme should be used.

To enable dark mode add `enable-dark-mode` to body class attribute.

You can test this feature by clicking on this two buttons. Must have dark mode enabled system wide!

If you don't have dark mode enabled you can view this video that demonstrates how it works.
