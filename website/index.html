Spartan UI KIT

Small UI kit ~6kb in size (without filtering and sorting) for application development.

Default font is Roboto Latin-ext implemented into css as @import.

Icons can be utf-8 characters ➜ copypastecharacter.com.

https://github.com/mitjafelicijan/spartan

View source for details

Why yet another UI kit?

Why develop yet another UI kit? Simply just to make my life easier. I usually need only some basic boilerplate kit and then I add styles accordingly. With this project I am trying to take back control by providing as minimal experience as possible so you can add your own styling on top of it and have clean interfaces that can be changed with couple of lines of code.

  • By default grid system is 960px in width and shrinks with page width.
  • Wrapper sets max width size of containers. If no .wrapper is present everything expands to 100% of page.
  • You can resize wrapper by changing .wrapper {max-width: 1200px}.
  • By default this grid is 12 column based (col-[1..12]). But is also supports tablet an mobile responsive option. Mobile option sets every column to 100%. Option in the middle is set between mobile and desktop. You can use it by setting additional class attribute (col-[1..12]-md).

Helper classes

Class name Description
.leftAligns text to left
.rightAligns text to right
.centerAligns text to center
.left-mdAligns text to left on medium screens
.right-mdAligns text to right on medium screens
.center-mdAligns text to center on medium screens
.left-smAligns text to left on mobile devices
.right-smAligns text to right on mobile devices
.center-smAligns text to center on mobile devices
.topAligns text to top
.middleAligns text to middle
.bottomAligns text to bottom
.hideHides element
.showShows element
.hide-mdHides element on medium screens
.show-mdShows element on medium screens
.hide-smHides element on mobile devices
.show-smShows element on mobile devices
.full-widthExpands element to parent
.empty-labelSimulates label on top of element
.borderAdds 2px border around of element and adds padding
